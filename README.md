# react-hot-starter

Hot starting point for React demos. Heavily based on [react-hot-loader](https://github.com/gaearon/react-hot-loader).

## Usage

1. npm install
2. npm start
3. Surf to `http://localhost:8080/webpack-dev-server/bundle`
4. Code like mad
5. ???
6. Profit
